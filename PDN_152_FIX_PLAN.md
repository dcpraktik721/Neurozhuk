## Контрольный чеклист перед следующим redeploy

- `npm run lint`
- `npm run build`
- Проверить `/auth/register`: checkbox не предустановлен, есть ссылки `/terms`, `/consent`, `/privacy`.
- Проверить `/auth/login`: есть notice и ссылка `/privacy`.
- Проверить footer: есть `/privacy`, `/consent`, `/terms`.
- Проверить `/privacy`: нет утверждения о подключенной аналитике, если она не подключена.
- Проверить `/api/health`: 200.
- Проверить headers на production после деплоя: HSTS, X-Frame-Options, Referrer-Policy, Permissions-Policy.
