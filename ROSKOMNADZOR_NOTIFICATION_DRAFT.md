# Черновик техсводки для уведомления Роскомнадзора

Дата подготовки: 3 мая 2026 г.

Этот файл подготовлен по фактическому коду проекта `neurozhuk.ru`. Он не заменяет юридическое уведомление: реквизиты оператора, регион базы данных Supabase и ответственный за обработку ПДн должны быть заполнены владельцем сайта до подачи.

## 2. Цели обработки

| Цель | Фактический процесс на сайте |
| --- | --- |
| Регистрация и ведение учетной записи | `/auth/register`, Supabase Auth, таблица `profiles` |
| Авторизация и поддержание сессии | `/auth/login`, Supabase auth cookies, middleware refresh |
| Сохранение игрового прогресса | `POST /api/sessions`, таблица `game_sessions` |
| Отображение статистики и прогресса | `GET /api/stats`, `GET /api/sessions`, таблицы `user_stats`, `game_sessions` |
| Достижения | `GET/POST /api/achievements`, таблица `achievements` |
| Обратная связь | `mailto:levart72@mail.ru` в footer, FAQ, policy/terms |
| Техническая диагностика и безопасность | Логи приложения, логи Timeweb Cloud, auth/session checks |

## 3. Категории субъектов

| Категория | Описание |
| --- | --- |
| Зарегистрированные пользователи | Пользователи, создающие аккаунт для сохранения прогресса |
| Дети/несовершеннолетние пользователи | Пользователи с возрастной группой `child`; по текстам сайта аккаунт должен создаваться родителем/представителем |
| Родители/законные представители | Лица, создающие аккаунт или обращающиеся по вопросам ребенка |
| Пользователи без аккаунта | Посетители публичных страниц и игры без сохранения прогресса |
| Обращающиеся по email | Пользователи, направившие письмо на контактный адрес |

## 4. Категории персональных данных

| Категория данных | Состав |
| --- | --- |
| Идентификационные данные аккаунта | Email, Supabase user id |
| Профиль | Имя/никнейм при указании, возрастная группа `child/adult`, дата создания/обновления |
| Auth-данные | Пароль передается в Supabase Auth; в прикладной БД проекта пароль не хранится |
| Игровые данные | Очки, правильные ответы, попытки, точность, длительность, режим, ранг, уровень, combo, даты сессий |
| Агрегированная статистика | Количество сессий, лучший результат, средние показатели, streak, общее время игры |
| Достижения | Ключ достижения, дата получения |
| Технические данные | IP-адрес, дата/время запроса, URL, User-Agent и статус ответа могут храниться в логах инфраструктуры |
| Обращения | Email отправителя, имя при указании, текст обращения, технические email-заголовки |

## 10. Меры защиты

| Мера | Фактическое состояние |
| --- | --- |
| HTTPS | `https://neurozhuk.ru` работает |
| Supabase RLS | В `supabase/schema.sql` включен RLS для `profiles`, `game_sessions`, `achievements`, `user_stats` |
| Доступ только к своим данным | Policies используют `auth.uid()` |
| Server-side guard | `/dashboard` и `/progress` закрыты middleware redirect |
| Auth-first API | `/api/sessions`, `/api/stats`, `/api/achievements` сначала проверяют пользователя |
| Runtime validation | `src/lib/validation/api.ts` валидирует игровые payload |
| Секреты | В репозитории используются только `NEXT_PUBLIC_*`; сервисный ключ Supabase в коде не найден |
| Security headers | Добавлены HSTS, X-Content-Type-Options, X-Frame-Options, Referrer-Policy, Permissions-Policy |
| Пароли | В прикладных таблицах не хранятся; регистрация/вход через Supabase Auth |

## 11. Что собирается через сайт

| Точка | Данные |
| --- | --- |
| `/auth/register` | Email, пароль, имя/никнейм при указании, возрастная группа, согласие на ПДн |
| `/auth/login` | Email, пароль, auth cookies |
| `/play` без аккаунта | Локальный highscore и настройка музыки в браузере; серверное сохранение не выполняется |
| `/play` с аккаунтом | Игровая сессия отправляется в `/api/sessions`; достижения проверяются через `/api/achievements` |
| `/dashboard`, `/progress` | Чтение профиля, статистики, истории сессий и достижений |
| `mailto:` | Email отправителя и текст обращения |

## 12. Cookie и аналитика

| Компонент | Статус |
| --- | --- |
| Supabase auth cookies | Технически необходимые cookies для авторизации |

| LocalStorage music flag | Локальная настройка музыки |
| Яндекс.Метрика / GA / GTM | В коде и live HTML не обнаружены |
| Сторонние script-src | В live HTML не обнаружены |
