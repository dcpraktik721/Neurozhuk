Эта игра разработана как школьный проект на конкурс
«Шаг в будущее, ЮНИОР». Автор: Корытников Лев.

───────────────────────────────────────────────────────
ПЕРЕД ПЕРВЫМ ЗАПУСКОМ
───────────────────────────────────────────────────────

На вашем компьютере должен быть установлен Node.js
версии 18 или новее.

  1. Откройте сайт https://nodejs.org
  2. Скачайте версию LTS (рекомендуемая)
  3. Установите как обычную программу
  4. Перезагрузите компьютер

Проверить установку можно так:
  • Windows: откройте «Командную строку» и введите node --version
  • macOS/Linux: откройте Терминал и введите node --version
  • Если видите номер версии (например v20.11.0) — всё готово

───────────────────────────────────────────────────────
🪟  WINDOWS
───────────────────────────────────────────────────────

  1. Распакуйте папку с игрой куда удобно
     (например на Рабочий стол)
  2. Откройте папку с игрой
  3. Дважды кликните на файл «Запустить-игру.bat»
  4. Откроется чёрное окно — это нормально, не закрывайте его
  5. При первом запуске установка займёт 2-3 минуты
  6. Браузер откроется автоматически на странице игры
  7. Чтобы закрыть игру — закройте чёрное окно или нажмите Ctrl+C

  ⚠️ Если Защитник Windows блокирует:
     Нажмите «Подробнее» → «Выполнить в любом случае»

  ⚠️ Если SmartScreen ругается на неизвестный издатель:
     Это нормально для школьного проекта. Файл безопасен.
     Нажмите «Подробнее» → «Выполнить в любом случае»

───────────────────────────────────────────────────────
🍎  macOS
───────────────────────────────────────────────────────

  1. Распакуйте папку с игрой
  2. Откройте папку с игрой
  3. Дважды кликните на «Запустить-игру.command»
  4. Откроется Терминал — не закрывайте его
  5. При первом запуске установка займёт 2-3 минуты
  6. Браузер откроется автоматически
  7. Чтобы закрыть игру — закройте Терминал или нажмите Ctrl+C

  ⚠️ Если macOS говорит «Не удаётся открыть, разработчик не определён»:
     - Кликните на файл правой кнопкой
     - Выберите «Открыть»
     - В диалоге снова нажмите «Открыть»

  ⚠️ Если ничего не происходит при двойном клике:
     - Откройте Терминал (Программы → Утилиты → Терминал)
     - Перетащите файл «Запустить-игру.command» в окно Терминала
     - Нажмите Enter

───────────────────────────────────────────────────────
🐧  LINUX
───────────────────────────────────────────────────────

  1. Откройте терминал в папке с игрой
  2. Выполните: chmod +x Запустить-игру.sh
  3. Запустите: ./Запустить-игру.sh
  4. При первом запуске установка займёт 2-3 минуты
  5. Браузер откроется автоматически
  6. Чтобы закрыть игру — нажмите Ctrl+C в терминале

───────────────────────────────────────────────────────
ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК
───────────────────────────────────────────────────────

🔴 «node не найден» / «node is not recognized»
   → Установите Node.js с https://nodejs.org и перезагрузитесь

🔴 «npm install» зависает
   → Проверьте интернет, попробуйте снова
   → Закройте окно и запустите лаунчер заново

🔴 Браузер не открылся автоматически
   → Откройте вручную: http://localhost:3000

🔴 Порт 3000 занят
   → Закройте другие программы или перезагрузите компьютер

🔴 Ошибка «EACCES» на macOS/Linux
   → В терминале выполните: chmod +x Запустить-игру.command
     (или Запустить-игру.sh для Linux)

🔴 Кракозябры вместо русских букв в чёрном окне Windows
   → Это не страшно, игра всё равно запустится
   → Дождитесь открытия браузера

КАК ИГРАТЬ

  Esc                         — пауза
